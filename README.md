# Secret Message Encoder

A small Flask web app for encoding and decoding messages with a simple Caesar cipher.

## Features

- Encode and decode text with a basic shift cipher
- Simple web interface using Flask
- Static CSS served from `static/style.css`

## Setup

1. Install dependencies:

```bash
pip install -r requirements.txt
```

2. Run the app:

```bash
python app.py
```

3. Open your browser at:

```text
http://127.0.0.1:5000
```
